## Prose and inline marks

Body text is set in Lora with real *italics*, **bold**, and `inline code`
that sits inside a sentence without jumping out of the line. Links like
[the Leptos book](https://book.leptos.dev) get a quiet underline that
sharpens on hover.

A hard\
break splits a line inside one paragraph.

Press <kbd>ctrl</kbd>+<kbd>c</kbd> to copy, and lowercase tags such as
<abbr title="Abstract Syntax Tree">AST</abbr> pass straight through with
their children still parsed as markdown.

### Lists

Unordered, for shapes of things:

- KV stores meaning
- deployed code owns presentation
- caches are purged, never rebuilt

Ordered, starting mid-sequence the way real numbered digressions do:

3. parse at publish time
4. render at request time
5. purge exactly what changed
