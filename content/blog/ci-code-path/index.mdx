# Code rides a deploy

The widget below is not part of the site's shared vocabulary — it lives in
`components.rs` right next to this file (ADR-0004). Pushing the two together
is a mixed commit: the pipeline worker parks the post as pending, fires the
publish workflow, and CI builds, checks the size budget, deploys both
workers, and only then drains the post through `/publish`.
